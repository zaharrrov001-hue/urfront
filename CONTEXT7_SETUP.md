## Настройка в Cursor

### ✅ Автоматическая установка (РЕКОМЕНДУЕТСЯ)

Конфигурация уже установлена автоматически! Просто перезапустите Cursor.

Если нужно переустановить, выполните:

```bash
./setup-mcp.sh
```

### Способ 1: Через настройки Cursor

1. Откройте Cursor Settings (Cmd/Ctrl + ,)
2. Найдите раздел "MCP Servers" или "Extensions"
3. Добавьте конфигурацию для Context7 (см. файл `mcp-config.json` в проекте)

### Способ 2: Вручную через конфигурационный файл

Конфигурация находится в `~/.cursor/mcp.json` и уже настроена автоматически.

Если нужно настроить вручную, используйте файл `mcp-config.json` из проекта как образец.

## Troubleshooting

Если Context7 не работает:

1. Проверьте, что API ключ правильный
2. Убедитесь, что MCP сервер запущен
3. Перезапустите Cursor
4. Проверьте консоль на наличие ошибок
